# 宿舍管理系统简介 java703

## 项目概述

宿舍管理系统是基于SpringBoot和Vue的Web应用，旨在提供宿舍管理的全面解决方案。该系统集成了前端代码、后端代码、数据库以及调试文档，确保项目的高效运行和便捷维护。

## 技术栈

- **后端：**
  - Java
  - Spring Boot
  - MyBatis-Plus
  - Maven
  - MySQL
- **前端：**
  - Vue

## 系统角色

系统定义了三种用户角色：

- 系统管理员
- 宿舍管理员
- 学生

### 系统管理员功能权限

- 首页查看
- 用户管理（学生和宿管）
- 宿舍管理（楼宇和房间）
- 信息管理（公告和报修信息）
- 申请管理（调宿）
- 访客管理
- 水电费管理
- 卫生管理
- 个人信息管理

### 宿舍管理员功能权限

- 首页查看
- 用户管理（学生）
- 宿舍管理（楼宇和房间）
- 信息管理（报修信息）
- 申请管理（调宿）
- 访客管理
- 卫生管理
- 个人信息管理

### 学生功能权限

- 首页查看
- 我的宿舍
- 申请调宿
- 报修申请
- 水电费管理
- 卫生检查
- 个人信息管理
